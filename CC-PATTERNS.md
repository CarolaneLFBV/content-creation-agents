# CC-PATTERNS.md - Retention & Engagement Patterns

Patterns éprouvés pour maximiser rétention, engagement et viralité.

## Pattern Philosophy

```yaml
core_principle: "Attention is earned every second"
retention_goal: ">70% watch time"
engagement_goal: ">5% engagement rate"
viral_threshold: "Share rate > comment rate"
```

---

## Hook Patterns (0-3 seconds)

### Pattern: Curiosity Gap
```yaml
pattern: curiosity-gap
mechanism: "Crée un vide d'information que l'audience veut combler"
power: 9/10
risk: "Clickbait si pas de payoff"

templates:
  - "Ce que personne ne vous dit sur [topic]"
  - "Le secret que [authority] ne veut pas que vous sachiez"
  - "J'ai découvert pourquoi [common problem]"
  - "Voilà ce qui se passe vraiment quand [action]"
  - "La vraie raison pour laquelle [phenomenon]"

execution:
  visual: "Expression intriguée / reaction shot"
  tone: "Confidentiel, conspirateur"
  energy: "Mid-high"
  follow_up: "DOIT livrer la réponse (pas de bait)"
```

### Pattern: Controversial Statement
```yaml
pattern: controversial
mechanism: "Provoque réaction émotionnelle immédiate (accord/désaccord)"
power: 9/10
risk: "Polarisant, peut backfire"

templates:
  - "[Opinion impopulaire] et je ne m'en excuse pas"
  - "Arrêtez de [popular advice]"
  - "[Common belief] est un mensonge"
  - "Je vais me faire détester mais [truth bomb]"
  - "Pourquoi je refuse de [norm]"

execution:
  visual: "Direct camera, confiant"
  tone: "Assuré, pas agressif"
  energy: "High"
  follow_up: "Arguments solides pour supporter"
```

### Pattern: Story Tease
```yaml
pattern: story-tease
mechanism: "Ouvre une boucle narrative que l'audience veut voir fermée"
power: 9/10
risk: "Must deliver satisfying ending"

templates:
  - "J'ai tout perdu quand [setup]..."
  - "Ce matin, j'ai reçu un message qui [tease]"
  - "Je n'aurais jamais dû [action], voilà ce qui s'est passé"
  - "La pire erreur de ma vie"
  - "Comment je suis passé de [bad] à [good]"

execution:
  visual: "Emotif, vulnérable ou shocked"
  tone: "Personnel, authentique"
  energy: "Variable selon émotion"
  follow_up: "Histoire complète avec arc narratif"
```

### Pattern: Direct Value Promise
```yaml
pattern: value-promise
mechanism: "Promesse claire de bénéfice en échange d'attention"
power: 7/10
risk: "Generic si mal exécuté"

templates:
  - "3 façons de [benefit] en [time]"
  - "Comment j'ai [achievement] sans [common method]"
  - "Le framework que j'utilise pour [result]"
  - "[Number] erreurs qui [negative consequence]"
  - "Fais ça si tu veux [desired outcome]"

execution:
  visual: "Liste visuelle ou démonstration"
  tone: "Expert, helpful"
  energy: "Mid"
  follow_up: "Livrer exactement ce qui est promis"
```

### Pattern: Pattern Interrupt
```yaml
pattern: interrupt-shock
mechanism: "Brise les attentes et force l'attention"
power: 8/10
risk: "Peut paraître gimmicky"

templates:
  - "STOP. Avant de scroller..."
  - "[Action inattendue] + maintenant que j'ai ton attention"
  - "[Son/visuel surprenant] + ok, parlons de..."
  - "Tu vas me détester pour ça"
  - "[Geste/action bizarre]"

execution:
  visual: "Mouvement rapide, changement soudain"
  tone: "Urgent, direct"
  energy: "High burst"
  follow_up: "Transition rapide vers contenu value"
```

### Pattern: Identity Call-Out
```yaml
pattern: identity-callout
mechanism: "Interpelle directement un groupe spécifique"
power: 8/10
risk: "Exclus ceux qui ne s'identifient pas"

templates:
  - "Si tu es [identity], tu dois savoir ça"
  - "Les [group] vont comprendre"
  - "POV: tu es [specific situation]"
  - "Ceci est pour les [group] qui [struggle]"
  - "Tag quelqu'un qui [behavior]"

execution:
  visual: "Direct, inclusif"
  tone: "Comme un ami qui comprend"
  energy: "Mid, relatable"
  follow_up: "Contenu ultra-relevant pour ce groupe"
```

## Retention Patterns (Mid-Content)

### Pattern: Open Loop
```yaml
pattern: open-loop
mechanism: "Tease information future pour maintenir l'attention"
power: 9/10
placement: "Every 15-30 seconds"

techniques:
  verbal:
    - "...mais avant de te dire ça, il faut que tu comprennes..."
    - "...et le plus fou c'est ce qui s'est passé après..."
    - "...je vais te révéler [thing] dans une seconde, mais d'abord..."
    - "...attends de voir ce qui arrive..."

  visual:
    - "Text overlay teasing what's coming"
    - "Quick flash of end result"
    - "Progress bar / countdown"

  structural:
    - "Promise in hook → Deliver at 70%"
    - "Multiple mini-reveals throughout"
    - "Delayed payoff technique"

execution_rules:
  - "Open max 2-3 loops simultaneously"
  - "Close loops before opening new ones (mostly)"
  - "Final loop = main hook payoff"
  - "Satisfying closure is NON-NEGOTIABLE"
```

### Pattern: Pattern Interrupt (Mid)
```yaml
pattern: mid-interrupt
mechanism: "Reset l'attention qui diminue naturellement"
power: 8/10
frequency: "Every 7-15 seconds"

techniques:
  visual:
    - "Jump cut"
    - "Angle change"
    - "B-roll insertion"
    - "Zoom in/out"
    - "Text pop"
    - "Color shift"

  audio:
    - "Music change"
    - "Sound effect"
    - "Voice modulation"
    - "Silence beat"

  content:
    - "Tangent then return"
    - "Quick joke"
    - "Self-interruption"
    - "Audience callout"

timing_guide:
  short_form: "Every 5-10s"
  mid_form: "Every 10-20s"
  long_form: "Every 20-45s"
```

### Pattern: Value Stacking
```yaml
pattern: value-stack
mechanism: "Chaque segment ajoute de la valeur, créant obligation de continuer"
power: 8/10

structure:
  segment_1: "Quick win (easy value)"
  segment_2: "Deeper insight (builds on #1)"
  segment_3: "Advanced tip (reward for staying)"
  bonus: "Secret/exclusive (final payoff)"

techniques:
  - "Numérotation visible (1/3, 2/3, 3/3)"
  - "Escalation de valeur ('et ça c'est rien comparé à...')"
  - "Callback to previous points"
  - "Promise of best at end"

verbal_bridges:
  - "Ok ça c'était le basique, maintenant le vrai game..."
  - "Et là où ça devient intéressant..."
  - "Mais le tip qui change tout..."
  - "Ce que les pros ne disent jamais..."
```

### Pattern: Story Peaks
```yaml
pattern: story-peaks
mechanism: "Pics émotionnels qui renouvellent l'investissement"
power: 9/10

emotional_peaks:
  tension: "Build anxiety/anticipation"
  revelation: "Surprise/twist moment"
  victory: "Success/achievement"
  vulnerability: "Raw honesty/failure"
  humor: "Comedic relief"

placement:
  - "Major peak at 70-80% of content"
  - "Mini peaks every 20-30%"
  - "Closing peak before CTA"

construction:
  buildup: "3-5 seconds of rising tension"
  peak: "1-2 second climax"
  release: "2-3 seconds aftermath"
  bridge: "Transition to next segment"
```

## Engagement Patterns

### Pattern: Comment Bait
```yaml
pattern: comment-bait
mechanism: "Crée une raison irrésistible de commenter"
power: 8/10

techniques:
  opinion_ask:
    - "Team [A] ou Team [B] ?"
    - "Tu es d'accord ou pas ?"
    - "Dis-moi en commentaire si..."

  fill_blank:
    - "Complète: [start of sentence]..."
    - "Le premier mot qui te vient c'est..."

  challenge:
    - "Prouve-moi que j'ai tort"
    - "Change my mind"
    - "Quelqu'un peut m'expliquer..."

  community:
    - "Tag quelqu'un qui..."
    - "Si tu vois ça, commente [word]"
    - "Les [group] comprendront"

placement:
  - "Early comment bait (0-30s) = boost algorithm"
  - "Mid content = engagement spike"
  - "End = CTA complement"
```

### Pattern: Save Trigger
```yaml
pattern: save-trigger
mechanism: "Donne une raison de sauvegarder pour plus tard"
power: 9/10 (especially Instagram)

techniques:
  utility:
    - "Sauvegarde pour quand tu auras besoin"
    - "C'est le genre de truc que tu vas vouloir retrouver"

  density:
    - "Trop d'info pour retenir"
    - "Liste/checklist format"
    - "Step-by-step guide"

  reference:
    - "Template réutilisable"
    - "Framework applicable"
    - "Reminder pour plus tard"

content_types_high_save:
  - "Tutorials / How-to"
  - "Lists / Resources"
  - "Templates / Frameworks"
  - "Checklists"
  - "Quotes / Inspiration"
```

### Pattern: Share Trigger
```yaml
pattern: share-trigger
mechanism: "Crée une raison de partager avec d'autres"
power: 10/10 (viral key)

triggers:
  identity_signal:
    - "Content that says something about the sharer"
    - "POV that resonates with a group"
    - "Achievement/transformation worth showing"

  utility_share:
    - "This would help my friend [X]"
    - "My [relation] needs to see this"
    - "Tag someone who [situation]"

  emotional_share:
    - "This made me feel [emotion]"
    - "I've never felt so seen"
    - "This is too good not to share"

  controversy_share:
    - "Look at this hot take"
    - "Do you agree with this?"
    - "This is going to upset people"

content_types_high_share:
  - "Relatable struggles"
  - "Controversial opinions"
  - "Useful resources"
  - "Emotional stories"
  - "Humor/memes"
```

## Viral Patterns

### Pattern: Trend Hijack
```yaml
pattern: trend-hijack
mechanism: "Utilise momentum existant avec angle unique"
power: Variable (timing-dependent)

execution:
  find: "Trending sound/format/topic"
  adapt: "Apply to your niche"
  differentiate: "Add unique twist"
  time: "Execute within 24-72h of emergence"

success_factors:
  - "Trend fit with brand (must feel natural)"
  - "Unique angle (not copy)"
  - "Quality execution (no rush jobs)"
  - "Timing (early = better)"

risk_mitigation:
  - "Only jump on brand-fit trends"
  - "Add genuine value, not just participation"
  - "Avoid forced/cringe adaptations"
```

### Pattern: Reaction Content
```yaml
pattern: reaction
mechanism: "Piggyback sur contenu existant populaire"
power: 8/10

types:
  duet_stitch:
    - "React to viral content"
    - "Add expertise/perspective"
    - "Challenge or support claim"

  commentary:
    - "Analyze trending topic"
    - "Expert take on news"
    - "Counter-opinion"

success_factors:
  - "Original source is viral/trending"
  - "Your take adds genuine value"
  - "Posted within trend window"
```

### Pattern: Series Viral
```yaml
pattern: series-viral
mechanism: "Chaque épisode drive vers le suivant"
power: 9/10 (compounding)

structure:
  naming: "Part 1/X, Episode X, Day X"
  cliffhanger: "End each with tease for next"
  progression: "Clear value escalation"
  accessibility: "Each works standalone too"

viral_mechanics:
  - "Binge-watch behavior"
  - "Anticipation building"
  - "Comment asking for next"
  - "Share to friends to catch up"
```

## CTA Patterns

### Pattern: Soft CTA
```yaml
pattern: soft-cta
mechanism: "CTA intégré naturellement dans le contenu"
power: 7/10 (high conversion, low reach impact)

techniques:
  - "Si tu veux plus de [topic], tu sais quoi faire"
  - "J'en parle plus sur [platform]"
  - "Comment si tu veux la suite"
  - "Profile pour [deeper content type]"

placement: "Mid-content or natural break"
frequency: "1-2 per content max"
```

### Pattern: Hard CTA
```yaml
pattern: hard-cta
mechanism: "Demande directe d'action"
power: 6/10 (lower engagement but clear)

techniques:
  - "Abonne-toi si [condition]"
  - "Like et partage si ça t'a aidé"
  - "Lien en bio pour [offer]"
  - "Commente [word] pour recevoir [thing]"

placement: "End of content only"
frequency: "1 per content max"
```

### Pattern: Embedded CTA
```yaml
pattern: embedded-cta
mechanism: "CTA qui fait partie de la valeur"
power: 9/10 (best of both worlds)

techniques:
  - "Pour appliquer ça, étape 1: follow pour les updates"
  - "Sauvegarde cette vidéo pour référence"
  - "Envoie ça à quelqu'un qui a besoin"
  - "Part 2 dispo sur mon profile"

principle: "CTA = Natural next step, not interruption"
```

## Pattern Combinations

### High Retention Stack
```yaml
combination: retention-stack
sequence:
  0-3s: curiosity-gap | story-tease
  3-10s: value-promise | pattern-interrupt
  10-30s: open-loop + value-stack
  30-60s: story-peaks + mid-interrupt
  60s+: payoff + embedded-cta

expected_retention: ">70%"
```

### Viral Stack
```yaml
combination: viral-stack
elements:
  hook: controversial | identity-callout
  mid: share-trigger + comment-bait
  end: cliffhanger + soft-cta

expected_share_rate: ">3%"
```

### Educational Stack
```yaml
combination: edu-stack
sequence:
  hook: value-promise + curiosity-gap
  body: value-stack + save-trigger
  end: bonus-value + hard-cta

expected_save_rate: ">5%"
```

## Anti-Patterns (Avoid)

```yaml
anti_patterns:
  - name: "Slow Start"
    problem: "Content before hook"
    fix: "Hook in first 1s, ALWAYS"

  - name: "Broken Promise"
    problem: "Hook ≠ Content"
    fix: "Deliver exactly what hook promises"

  - name: "Energy Death"
    problem: "Monotone throughout"
    fix: "Pattern interrupts + energy modulation"

  - name: "Loop Abandonment"
    problem: "Open loops never closed"
    fix: "Track and close all loops"

  - name: "CTA Spam"
    problem: "Too many asks"
    fix: "1 clear CTA, embedded naturally"

  - name: "Value Delay"
    problem: "Making people wait too long"
    fix: "Value in first 10s, escalate from there"
```
